Zenn Markdown

Plugin is enabled. Markdown files will be rendered with Zenn-style syntax.

Supported Syntax

  • :::message - Message box
  • :::message alert - Alert box (red)
  • :::details title - Collapsible section
  • @[youtube](videoId) - YouTube embed
  • @[tweet](url) - Twitter/X embed
  • @[github](url) - GitHub file embed
  • @[gist](url) - GitHub Gist embed
  • @[card](url) - Link card
  • @[slideshare](key) - SlideShare embed
  • @[speakerdeck](key) - Speaker Deck embed
  • @[codepen](url) - CodePen embed
  • @[codesandbox](url) - CodeSandbox embed
  • @[stackblitz](url) - StackBlitz embed
  • @[figma](url) - Figma embed
  • @[mermaid](code) - Mermaid diagram
  • $formula$ - Inline math (KaTeX)
  • $$formula$$ - Block math (KaTeX)
  • ![](image.jpg =300x200) - Image with size
